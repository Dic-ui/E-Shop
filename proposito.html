Propósito del Proyecto E-Shop

E-Shop UNAD es una plataforma académica de comercio electrónico que busca mejorar la experiencia de compra en línea y optimizar la gestión de productos, pedidos y usuarios. El proyecto fue desarrollado como caso de estudio del curso Calidad de Software.

Objetivos Específicos

  • Analizar y aplicar estándares de calidad de software ISO/IEC 25010.
  • Simular procesos de autenticación, compra y soporte dentro del sitio.
  • Implementar pruebas automatizadas para validar la funcionalidad del sistema.
  • Evaluar el comportamiento del sistema en condiciones de uso realista.

Alcance

El alcance del proyecto abarca la simulación de un sistema completo de comercio electrónico sin base de datos, implementando módulos de autenticación, catálogo, carrito, pago y soporte, dentro de un solo archivo HTML. Las pruebas permiten evaluar la interacción, rendimiento y estabilidad visual del sistema.
